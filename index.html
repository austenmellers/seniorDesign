Fault Tolerant Controller

Problem Statement

The purpose of the fault predictor project is to determine the likelihood of a motor failing. This project needs to measure relevant values, process the data, and if necessary, control the motor so that it will not fail.

Deliverables

Operation Manual
Final Report

Team Goals

The team aims to solve the problem set out in the problem statement in a safe, efficient, and economical manner, as well as publish the results in a conference or journal paper.

Project Description

The fault tolerant controller is a senior design project by Team #9 at the FAMU-FSU College of Engineering. It aims to:

  • Monitor motor with sensors
  • Analyze sensor data
  • Determine a fault model
  • Output a 0 to 1 likelihood that the motor will fault
  • Display data on a user interface

The fault tolerant controller is sponsored by General Atomics, and aims to predict when a motor will fault before it happens. The goal of the team is to output a real-time probability in the form of a number between 0 and 1 that the motor will fault, to a user interface on a website or a tablet
